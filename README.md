# NodeJS Gemini API 🚀

A simple Node.js API that connects to **Google Gemini AI** using the official `@google/genai` SDK.  
This project exposes an **Express API endpoint** that generates AI responses using the Gemini model.

---

## Features

- Node.js + Express REST API
- Google Gemini AI integration
- Environment variables using dotenv
- JSON request/response support
- Simple structure for easy integration with frontend apps

---

node main.js
